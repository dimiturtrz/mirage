# research/

Teacher-side grounding behind the `learning/` writeups — the cited raw material researched
before teaching a topic. Rougher than `learning/` (the distilled study version), but public:
it shows the sources and reasoning the understanding rests on.

Split of labor: **`learning/` is for me** (the human — distilled theory, my ramp into 3D);
**`research/` is the agent's** (cited deep-dives that ground the learning + the plan).

Format:
- `deep_dives/<date>_<topic>.md` — one focused, cited research pass per topic.
- `NOTICE_BOARD.md` — open questions + settled findings index + progress log.
- `summaries/` — cross-topic synthesis when it accumulates.

The circuit: **research (ground it → `research/`) → theory writeup (`learning/`) → quiz → sharpen the plan.**
